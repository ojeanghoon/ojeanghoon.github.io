# History of Java

### JAVA 버전

**JDK Alpha and Beta** 1995년

**JDK 1.0(JAVA 1)** 1996년 1월 23일

##### 가장 첫번째 버전, JDK 1.0.2 버전

**JDK 1.1** 1997년 2월 19일

##### AWT 개편, JDBC, RMI

**J2SE 1.2** 1998년 12월 8일

##### 코드명(Play area), 중심 클래스 Swing 그래픽 API, JIT 컴파일러, 자바모듈, JAVA IDL(CORBA), 컬렉션 시스템

**J2SE 1.3** 2000년 5월 8일

##### 코드명(KESTREL), 핫스팟 JVM포함, JNDI, JPDA, 자바사운드, Synthetic proxy

**J2SE 1.4** 2002년 2월 6일

##### 코드명(Merlin), 라이브러리 개선, 정규 표현식(Perl), 이미지 I/O, XML 파서, XSLT, JAXP, Preferences API

**J2SE 5.0** 2004년 9월 30일

##### 코드명(Tiger), for-each 문, 제네릭, 오토 박싱, Var-args

**자바 SE 6** 2006년 12월 11일

##### 코드명(Mustang), 버전 번호에서 .0 삭제 , JAXB, 어노테이션 도입, JDBC 4.0

**자바 SE 7** 2011년 7월 7일

##### 코드명(Dolphin), 압축 된 64bit 포인터, 이진 정수 리터럴, XML 및 유니코드에 대한 업스트림 업데이트

**자바 SE 8** 2014년 3월 18일

##### 람다 표현식 도입, Joda Time에서 영감을 얻은 새로운 날짜 및 시간 API

**자바 SE 9** 2017년 9월 21일

##### Project Jigsaw : Java Se 플랫폼 용 표준 모듈 시스템을 설계 및 구현

**자바 SE 10** 2018년 3월 20일

##### 추가 유니코드 언어 태그 확장, 루트 인증서, 대체 메모리 장치의 힙 할당, 기본 헤더 생성도구 javah를 제거, JDK 포리스트를 단일 저장소로 통합

**자바 SE 11** 2018년 9월 25일

##### 동적 클래스 파일 상수, 앱실론, 람다 매개 변수의 지역변수 구문, 오버헤드가 적은 힙 프로파일링, HTTP 클라이언트(standard), TLS 1.3

**자바 SE 12** 2019년 3월 19일

##### 셰넌 도어 , 마이크로 벤치 마크 스위트, 스위치 식, JYM 상수 APIM, AArch64포트 1개(2개 x), 기본 CDS아카이브
