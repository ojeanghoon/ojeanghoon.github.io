# Huffman Tree

### 텍스트 파일에 사용된 예시문 :

 "As far as Ukrainians are concerned, the war has been going on for eight years already,"



1. Java 코드를 통해 나온 결과값들을 바탕으로 각각의 문자에 대해 node를 만든다.
